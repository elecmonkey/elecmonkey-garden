## 关于 Next.js

Next.js 是一个基于 React 的框架，它提供了许多有用的功能，如：

- 服务器端渲染 (SSR)
- 静态站点生成 (SSG)
- 文件系统路由
- API 路由

## 代码示例

```javascript
// 这是一个简单的 React 组件
function Greeting({ name }) {
  return <h1>你好, {name}!</h1>;
}

export default Greeting;
```
