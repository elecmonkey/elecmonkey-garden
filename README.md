# Elecmonkey's Garden

[https://www.elecmonkey.com](https://www.elecmonkey.com)

个人技术博客，基于 [Next.js](https://nextjs.org)。

## 内容管理

博客文章存储在 `src/content/posts` 目录中，使用 Markdown 格式。每篇文章需要包含以下前置元数据：

```md
---
title: '文章标题'
date: 'YYYY-MM-DD'
description: '文章简短描述'
tags: ['标签1', '标签2']
author: 'Elecmonkey'
---

文章内容...
```
